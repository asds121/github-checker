## 8. 文档维护指南

### 8.1 添加新文档

1. 根据文档类型选择合适的目录
2. 使用相应的模板创建文档
3. 填写文档信息表格
4. 更新此索引文档

### 8.2 更新现有文档

1. 修改文档内容
2. 更新文档信息表格中的版本和日期
3. 在修订历史中添加变更记录
4. 更新此索引文档（如需要）

### 8.3 归档旧文档

1. 将文档移动到archive目录
2. 在索引中更新文档状态为"已归档"
3. 保留文档的访问链接
