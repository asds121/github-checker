## 🎯 项目概述

### 项目名称

GitHub Network Status Checker (github-checker-检测状态)

### 项目目标

开发一个极简的命令行工具，用于检测 GitHub 的访问状态，帮助用户判断当前网络环境是否适合进行 GitHub 相关操作。

### 目标用户

- 开发者
- 运维工程师
- 普通用户

### 技术栈

- **编程语言**：Python 3.6+
- **核心依赖**：requests
- **操作系统**：Windows、macOS、Linux

### 1. 核心检测功能

- ✅ GitHub 主页（github.com）检测
- ✅ GitHub API（api.github.com）检测
- ✅ HTTP 请求发送和响应处理
- ✅ 状态码检查（200 OK）
- ✅ 响应时间测量（毫秒级精度）

### 2. 快速测试模式

- ✅ 单轮检测流程
- ✅ 超时控制（默认8秒）
- ✅ 结果收集和汇总
- ✅ 状态判断逻辑
  - 所有目标成功且平均延迟 < 3000ms → good
  - 所有目标成功但平均延迟 ≥ 3000ms → warn
  - 部分目标成功 → warn
  - 所有目标失败 → bad
